# persona-distill 🎭

> 将任何人的写作风格、说话方式、知识体系蒸馏成可复用的 Persona Skill

## 是什么

persona-distill 是一个 OpenClaw Skill，通过"三步蒸馏法"把一个人的表达特征提取出来，打包成标准化的 Persona Skill，让 AI 能用这个人的风格说话。

## 三步蒸馏法

### Step 1：素材收集
提供目标人物的文字样本——聊天记录、文章、邮件、社交媒体帖子，越多越好。

### Step 2：风格提炼
通过快问快答提炼核心特征：
- 句式偏好（长句/短句/跳跃式）
- 口头禅和习惯用语
- 语气强度和情绪倾向
- 夸人/批评/犹豫时的表达模式

### Step 3：生成 Persona Skill
将提炼结果填入标准化模板，生成可直接安装的 Persona Skill。

## 安装

```bash
skillhub install persona-distill
```

## 使用

在对话中说：
- "帮我蒸馏一下我的风格"
- "把这个人的说话方式提取出来"
- "我想做一个自己的数字分身"

Skill 会引导你完成三步蒸馏流程，最终生成一个 `.skill` 文件。

## 生成的 Persona Skill 示例

蒸馏完成后会生成包含以下结构的 Skill：

```markdown
---
name: my-persona
description: "以用户的风格进行对话"
---

## 我怎么说话
- 能三个字说完绝不用十个字
- 句尾爱带"吧"，给自己留退路

## 我怎么想
- 先想"有可能不行"，再想"但试试也无妨"
- 不确定的事不把话说死

## 风格示例
**普通说法**：这个方案不错，但有一些风险需要考虑。
**我的说法**：方案还行，就是有点悬。
```

## 优点

- ✅ 无需编程，全程对话完成
- ✅ 5-10分钟出结果
- ✅ 支持多种素材（文章、聊天记录、邮件、口述）
- ✅ 没素材也能做（快问快答模式）
- ✅ 输出标准 Skill 包，可安装、可分享

## 局限性

- ❌ 只能捕捉「显性风格」（用词、句式、节奏），抓不住「隐性智慧」（深层思考方式、知识图谱）
- ❌ 素材质量决定输出质量——垃圾进垃圾出
- ❌ 快问快答模式精度有限，最好有真实素材
- ❌ 生成的 Skill 需要人工校验，不保证100%还原

## 适用场景

- 🎯 想让 AI 模仿自己的说话风格
- 📝 为公众号/社交媒体内容保持统一的语感
- 🤖 创建个人数字分身
- 📖 把导师/前辈的表达方式沉淀下来

## 技术细节

- 纯提示词驱动，无需 API Key
- 生成的 Persona Skill 兼容 OpenClaw / QClaw 生态
- 支持中英文风格提炼
